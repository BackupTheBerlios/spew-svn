Installation instructions

Source code
  1. Download source tarball.
  2. Untar tarball in temporary directory.
  3. Cd to extracted directory.
  4. Run ./configure.
  5. Run make.
  6. Run make install as root.
linux-ia32
linux-ia32
linux-parisc64
  1. Download binary tarball.
  2. Cd to /.
  3. Untar tarball as root (files go to /usr/local).
cygwinnt-ia32
  1. Download binary tarball.
  2. Install the cyginw32 system .
  3. Cd to installation directory.
  4. Unzip binary zipfile (files go to usr/local in installation directory).
  5. Make sure the cygwin1.dll installed with the Cygwin system is somewhere in your PATH.
  6. Add <installation directory>/usr/local/bin to your PATH.
